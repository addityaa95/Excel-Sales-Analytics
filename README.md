## Atliq Hardwares Sales & Profitability Analysis :

-**Project Overview :**

  **This project analyzes the sales performance and profitability of Atliq Hardwares, a global hardware company operating 
  across multiple countries and markets. The objective is to transform raw transactional data into meaningful business insights 
  using Excel’s advanced analytics stack, including Power Query, Power Pivot, and DAX.
  
  The project demonstrates the complete data analytics workflow: data extraction, transformation, modeling, analysis, and dashboard creation.**

## Objective :
- [x] Analyze company sales performance across different customers and regions 
- [x] Evaluate Gross Margin trends over fiscal years
- [x] Compare actual performance against targets
- [x] Identify high-performing and low-performing markets 
- [x] Build an efficient data model using Power Pivot
- [x] Create scalable and dynamic dashboards using Excel

 ## Key Insights:
- [x] Net Sales increased from ₹87.48M in FY2019 to ₹598.88M in FY2021 (304% growth)
- [x] India is the top-performing market in terms of total sales
- [x] Gross Margin % declined slightly, indicating rising operational costs
- [x] Several countries underperformed against targets, highlighting optimization opportunities
- [x] Strong global expansion trend across multiple markets
